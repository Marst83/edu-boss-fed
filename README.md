### 介绍
这是拉勾训练营的项目，有权限管理，课程管理，用户管理，广告管理四个模块

简单demo，较为粗糙

主要技术栈是Vue2.x + typescript

后端接口由拉勾提供，用户端的地址是 https://edufront.lagou.com/ 

课程上下架和视频观看，可以在用户端看到，但是用户端的网站不太稳定，可能会挂掉
